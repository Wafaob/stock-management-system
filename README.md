# Stock Management System

A simple web application created to practice HTML, CSS and JavaScript.

## Project Goal

The goal of this project is to build a simple application for managing products and stock quantities.

## Current Progress

- HTML structure created
- Stock table added
- Initial CSS styling added
- JavaScript development in progress

## Planned Features

- Add new products
- Search products
- Filter products by category
- Edit and delete products
- Display low-stock alerts
- Save data with localStorage

## Technologies

- HTML5
- CSS3
- JavaScript

- GitHub
